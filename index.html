Mass Automated Mail Assistant (MAMA)

This is my precious baby. Mass Automated Mail Assistant (MAMA) is an application made with VBA in Microsoft Excel and utilize Microsoft Outlook to send e-mails. MAMA is designed with the intention to send numerous e-mail to recipients individually. It can send 300 or more e-mail in mere minutes, which greatly improving work efficiency.

MAMA is specially created when I was working in a high security location that does not allow 3rd-party software. The challenge was I have to create a solution without using or needing any form of 3rd-party library or application other than the ones that was available on that computer. What was installed and available at that time was Microsoft Office and therefore, I have to make-do with what I have.

Student Attendance System (SAS)

As the project name suggest, its main job is to keep a record of student's attendance. SAS need to be well-optimized and maintained as it cater to more than 3000 students and the entire staff of the school. It is also capable of sending a notification to recipient(s) about the student's attendance.

This project is especially memorable because of an issue we encountered. Several user notify us that the system had "spun" for a very long time. All of the system was tested working and nothing was out of the ordinary which was puzzling. I, however, was even more intrigued by this issue and began to look deeper. I found out that it was caused by an extremely long SQL statement that excuted for 14 seconds. I informed my team and they agreed that that was the issue. Thus, I began to analze and re-write the SQL in attempt to improve its performance.

The new SQL statement took 1 ms to complete the operation and to our surprise, we did not hear any more issue about the system "spinning" for too long. This project serves as a reminder to not write lenghty SQL statement that execute for 14 seconds and to keep it simple whenever possible
